\# Excel Superstore Sales Dashboard



\## Project Overview

Interactive sales dashboard built on 9,994 rows of US retail data.



\## Tools Used

\- Microsoft Excel 2019

\- Pivot Tables, Pivot Charts, Slicers, Conditional Formatting



\## Key Insights

\- West region has highest sales ($725,457)

\- Technology is the most profitable category ($145,454 profit)

\- Consumer segment drives 50% of total orders

\- Texas has high sales but negative profit (-$25,729)



\## Dashboard Features

\- 4 interactive Pivot Charts

\- 3 slicers (Region, Category, Segment) filtering all charts simultaneously

\- 3 KPI metrics (Total Sales, Total Profit, Total Orders)
